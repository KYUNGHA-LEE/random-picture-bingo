# 그림보고 랜덤 빙고 게임 (random-picture-bingo)

선생님과 학생이 각자의 기기로 접속해 실시간으로 함께 진행하는 수업용 빙고 게임입니다. 선생님이 그림을 보여주면 학생이 자신의 빙고판에서 같은 단어를 찾아 클릭합니다. Firebase로 모든 화면이 즉시 동기화되며, 별도 설치 없이 웹 브라우저만으로 작동합니다.

## 주요 기능

- **실시간 동기화**: 출제·정답 처리·우승 판정이 모든 참여자 화면에 즉시 반영됩니다.
- **참여자별 무작위 빙고판**: 게임 시작 시 학생마다 서로 다른 빙고판이 생성됩니다.
- **빙고판 크기·우승 조건 설정**: 3×3 / 4×4 / 5×5 선택, 우승 빙고 수 지정.
- **이미지 업로드**: 파일명이 그대로 빙고 단어로 사용됩니다.
- **선생님 단독 입장 잠금**: 한 방에 선생님은 한 명만 입장할 수 있습니다.
- **방 분리**: 주소에 `?room=방이름`을 붙여 여러 학급을 동시에 운영할 수 있습니다.
- **우승 연출·효과음**: 색종이 애니메이션과 효과음을 제공합니다.

## 실행 방법

정적 사이트이므로 빌드 과정이 없습니다. Firebase Hosting·Vercel·GitHub Pages 등 정적 호스팅에 배포하거나, 로컬에서 간단한 웹 서버로 실행합니다.

```bash
python -m http.server 8000
```

> ES 모듈을 사용하므로 `file://`로 직접 열면 동작하지 않습니다. 반드시 웹 서버나 호스팅을 통해 접속하세요.
> Firebase를 직접 쓰려면 `firebase_config.js`의 설정값을 본인 프로젝트 정보로 교체하세요.

## 기술 스택

- HTML · CSS · JavaScript (Vanilla, ES Modules)
- Firebase Realtime Database (실시간 상태 동기화)
- Firebase Storage (이미지 저장)
- Web Audio API (효과음)
